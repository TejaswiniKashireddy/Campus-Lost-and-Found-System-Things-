Project Title:

Campus Lost and Found System

Problem:

Students often lose items on campus and usually post information in random social media groups, which makes it difficult for the information to reach the right person.

Solution:

This web application provides a centralized platform where students can report lost items or post found items so that owners can easily recover their belongings.

Technologies Used:

HTML

CSS

JavaScript

Python Flask

SQLite Database

Features:

User registration and login

Post lost items

Post found items

Search lost/found items

View personal posts

How to Run:
Install Python

Install Flask

Run app.py

Open http://127.0.0.1:5000 in browser
